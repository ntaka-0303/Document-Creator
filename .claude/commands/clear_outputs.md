# 出力クリアコマンド

`docs/<doc_type>/output` ディレクトリ内のファイルを削除。

## 対話フロー

### 1. ドキュメント種別選択

```
クリア対象を選択してください：

1. 提案書
2. 計画書
3. 要件定義書
4. To-Beワークフロー
5. 機能仕様書
all. すべて

番号 (1-5) または 'all':
```

### 2. 確認プロンプト

```
以下を削除します：
  - docs/<ディレクトリ>/output/*

よろしいですか？ (yes/no):
```

`yes` の場合のみ実行。

### 3. 削除実行

- `output/` 配下のすべてのファイルとサブディレクトリを削除
- `output/` ディレクトリ自体は残す
- `definitions/` や `00_inputs/` は削除しない
