# 評価指標（計画書）

## 評価観点

1. 実現可能性
   - 計画の実行可能性が適切に検討されているか。

2. わかりやすさ・構造性
   - 計画の全体像が理解しやすい構成になっているか。

3. 具体性・詳細度
   - タスク、スケジュール、リソース配分が具体的に示されているか。

4. リスク管理
   - 想定されるリスクと対策が明記されているか。

5. テンプレ準拠度
   - 章構成・見出しが大きく崩れていないか。

## スコアリング

- 各観点について 1〜5 点でスコアリングすること。
- 1: 計画として不十分
- 3: 一般的な計画書として許容できるレベル
- 5: 実行に移せる高品質なレベル

## 合格基準

- 以下を満たす場合に `pass` とする。
  - 総合評価（全観点の平均）が 4 以上
  - 「実現可能性」「具体性・詳細度」がいずれも 4 以上
